# The Adversarial Hook: Understanding Initial Access and Exploitation

## Outline
- Initial Access
- MITRE ATT&CK Framework
- Drive-by Compromise
- Fake CAPTCHA Campaign
- Social Engineering
- Post-Exploitation and Persistence

## Definition
The Adversarial Hook focuses on how attackers gain access to systems through exploitation, malicious websites, and social engineering techniques.

## MITRE ATT&CK Framework
The MITRE ATT&CK Framework is a knowledge base of adversarial tactics and techniques used during cyber attacks.

## Drive-by Compromise
A drive-by compromise occurs when a user visits a malicious or compromised website that silently delivers malware.

### Common Methods
- Injecting malicious code into websites
- Malvertising
- Cross-site scripting (XSS)
- Compromised cloud storage scripts

## Fake CAPTCHA Campaign
The Fake CAPTCHA campaign tricks users into executing malicious PowerShell commands disguised as verification steps.

### Techniques Used
- SEO poisoning
- Compromised domains
- PowerShell abuse
- MSHTA abuse

## Social Engineering
Attackers manipulate users into performing unsafe actions such as downloading malware or running commands.

## Post-Exploitation and Persistence
After gaining access, attackers attempt to maintain control through persistence techniques.

### Common Payloads
- Infostealers
- Remote Access Trojans (RATs)

### Persistence Methods
- Registry Run Keys
- DLL Sideloading
