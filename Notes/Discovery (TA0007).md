# Discovery (TA0007)

## Outline
- Discovery
- Discovery Taxonomy
- Network Service Discovery
- Account Discovery
- Defense Evasion
- Host and System Discovery

## Definition
Discovery techniques are used by attackers to gather information about systems, users, and networks after gaining access.

## Goal of Discovery
Attackers collect information to:
- Identify targets
- Map networks
- Discover accounts
- Detect security tools
- Prepare for lateral movement

## Discovery Taxonomy

### Identity and Access
- Account Discovery
- Permission Groups
- Password Policies

### Network and Infrastructure
- Network Service Discovery
- Cloud Infrastructure Discovery
- Domain Trust Discovery

### System State and Environment
- Process Discovery
- Software Discovery
- Virtual Machine Discovery

## T1046 Network Service Discovery
Attackers scan systems to identify:
- Open ports
- Running services
- Vulnerable software

### Common Tools
- Nmap
- Ping
- DNS queries

## T1087 Account Discovery
Attackers identify valid accounts for privilege escalation or lateral movement.

## Defense Evasion
Attackers may detect:
- Antivirus software
- Sandboxes
- Debuggers

## Key Points
- Discovery provides attackers with environmental awareness.
- Scanning and enumeration are common discovery activities.
- Monitoring unusual reconnaissance behavior is important.
