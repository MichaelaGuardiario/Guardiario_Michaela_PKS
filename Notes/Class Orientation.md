# Class Orientation

## Outline
- Course Introduction
- Grading Breakdown
- Certification Pathway
- Forum Grades
- Personal Knowledge System (PKS)

## Definition
The class orientation introduced the course structure, grading system, certifications, and project requirements. Students are expected to participate in practical cybersecurity activities and maintain a Personal Knowledge System (PKS) using Markdown files.

## Key Points
- Quizzes account for 20% of the grade.
- Forums and assignments involve Hack The Box and TryHackMe activities.
- Certifications can provide perfect scores for related assessments.
- The PKS project must be self-contained and written in Markdown format.
