# Initial Access (TA0001) - Valid Accounts (T1078)

## Outline
- Valid Accounts
- Credential Abuse
- Perimeter Bypass
- Attack Vector Funnel
- Initial Access Techniques

## Definition
Valid Accounts (T1078) is a MITRE ATT&CK technique where attackers use stolen or compromised credentials to gain access to systems and services.

## Credential Abuse
Attackers use valid usernames and passwords to bypass authentication systems.

### Common Targets
- VPNs
- Remote Desktop Protocol (RDP)
- Webmail Services
- Cloud Services

## Perimeter Bypass
Traditional defenses become ineffective when attackers successfully authenticate using legitimate credentials.

## Attack Vector Funnel
Attackers gather credentials through:
- Data breaches
- Phishing
- Weak passwords
- Third-party compromises

## Impact
Successful credential abuse can lead to:
- Unauthorized access
- Privilege escalation
- Lateral movement
- Data exfiltration

## Key Points
- Weak passwords are a major cybersecurity risk.
- Multi-factor authentication helps reduce credential attacks.
- Identity-based attacks are common in modern cyber threats.
