# Personal Knowledge System (PKS)

## Overview
This repository contains my Personal Knowledge System (PKS) notes for cybersecurity topics discussed during the course. The notes are written in Markdown format and organized by topic for easy navigation and review.

## Contents
- Cybersecurity Fundamentals
- Security Models and Threat Frameworks
- MITRE ATT&CK Techniques
- Initial Access
- Execution
- Persistence
- Discovery
- Career Opportunities in Cybersecurity

## Purpose
The purpose of this project is to build an organized and connected knowledge base using Markdown files that can be opened in GitHub or Obsidian.

## Tools Used
- Markdown (.md)
- GitHub
- Obsidian

## Notes
All files, images, and resources are stored locally to ensure the project is self-contained and free from broken links.
